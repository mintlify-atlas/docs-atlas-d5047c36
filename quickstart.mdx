This guide gets you from zero to your first tool call in under 2 minutes.

<Tip>
  All 48 utility tools work without any API key or network connection.
</Tip>

<Steps>
  <Step title="Install via Homebrew">
    ```bash
    brew install GustavoGutierrez/devforge/devforge
    ```

    This installs `devforge`, `devforge-mcp`, and `dpf` and creates a starter config at `~/.config/devforge/config.json`.

    See [Installation](/installation) for source build instructions and platform notes.
  </Step>

  <Step title="Verify the installation">
    Run `devforge-mcp` with the version flag to confirm it is installed correctly:

    ```bash
    devforge-mcp --version
    ```

    `devforge-mcp` is a stdio MCP server — it does not expose an HTTP port. When started by an MCP client, it reads JSON-RPC messages from stdin and writes responses to stdout.
  </Step>

  <Step title="Connect to an MCP client">
    Add the following entry to your MCP client configuration. For Claude Desktop this is `claude_desktop_config.json`; for Cursor it is your MCP settings file.

    ```json
    {
      "mcpServers": {
        "devforge": {
          "command": "/path/to/devforge-mcp",
          "args": [],
          "env": {}
        }
      }
    }
    ```

    Replace `/path/to/devforge-mcp` with the actual binary path. Run `which devforge-mcp` to find it.

    See [MCP client setup](/mcp-setup) for platform-specific instructions and tips for Claude Desktop, Cursor, and other clients.
  </Step>

  <Step title="Make your first tool call">
    Once your MCP client is connected, call `crypto_hash` to hash a string. This tool is stateless, requires no API key, and works offline — a reliable first test.

    ```json
    {
      "name": "crypto_hash",
      "arguments": {
        "input": "hello world",
        "algorithm": "sha256",
        "encoding": "hex"
      }
    }
    ```

    The tool returns the SHA-256 hex digest of the input string.
  </Step>

  <Step title="Try another tool">
    Call `text_uuid` to generate a random UUID. This tool takes no arguments:

    ```json
    {
      "name": "text_uuid",
      "arguments": {}
    }
    ```

    You should receive a freshly generated UUID v4 in the response.
  </Step>
</Steps>
