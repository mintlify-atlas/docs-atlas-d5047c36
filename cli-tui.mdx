The `devforge` binary launches an interactive [Bubble Tea](https://github.com/charmbracelet/bubbletea) terminal UI that gives you access to the full DevForge tool surface from the command line — no AI client required.

## Launch the TUI

```bash
devforge
```

That is the only command. The TUI starts immediately and presents the main menu.

<Note>
  The TUI reads the same config file as the MCP server (`~/.config/devforge/config.json`). Any Gemini API key you configure in one is available in the other.
</Note>

Every tool available via the MCP server is also reachable from the TUI. The 48 utility tools are grouped as follows:

<Note>
  The TUI and the MCP server expose the same tool surface — all 70+ tools are available in both interfaces. Any tool you can call from an AI agent you can also run interactively from the terminal.
</Note>
