DevForge MCP is a Go-powered MCP server and CLI/TUI that puts over 70 stateless developer tools directly into your AI agent or terminal. It covers image and video processing, layout analysis, design tokens, cryptography, HTTP utilities, data formatting, and code tools — all with zero database dependencies and no persistent state.

## How it works

<Steps>
  <Step title="Install DevForge">
    Install all three binaries (`devforge`, `devforge-mcp`, `dpf`) with a single Homebrew command or build from source.
  </Step>
  <Step title="Connect your AI client">
    Add the `devforge-mcp` binary to your MCP client config. Works with Claude Desktop, Cursor, and any stdio-compatible MCP client.
  </Step>
  <Step title="Call tools from your agent">
    Your AI agent can now call any of the 70+ tools by name. All tools return structured JSON — no hallucination, no ad-hoc code generation needed.
  </Step>
  <Step title="Use the CLI/TUI optionally">
    Run `devforge` for an interactive terminal UI to access the same tool surface without an AI client.
  </Step>
</Steps>

<Note>
  All non-AI tools work fully offline and require no API key. Only `generate_ui_image`, `ui2md`, and `suggest_color_palettes` require a Gemini API key, which you can configure at runtime.
</Note>
